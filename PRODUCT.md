# Product

## Register

product

## Users

นักศึกษาไทยที่กำลังหาที่ฝึกงาน — ใช้มือถือเป็นหลัก, เปิดดูระหว่างเรียน/พักเที่ยง, ต้องการข้อมูลจริงจากคนที่เคยฝึกงานมาก่อน รวมถึงพี่ๆ พนักงานจริงที่อยากแชร์ประสบการณ์การทำงานในบริษัท

Job to be done:
- **ผู้อ่าน:** ค้นหาและเปรียบเทียบที่ฝึกงาน/ทำงาน จากรีวิวจริงของคนที่เคยผ่านมา
- **ผู้เขียน:** แชร์ประสบการณ์ฝึกงาน/ทำงานจริง ให้รุ่นน้องได้ประโยชน์

## Product Purpose

พื้นที่ปลอดภัยให้นักศึกษาและคนทำงานแชร์ประสบการณ์ฝึกงาน/ทำงานจริง ไม่ fake ไม่ผ่านการคัดกรองจากบริษัท — รวบรวมรีวิวที่ปัจจุบันกระจัดกระจายใน Facebook groups, TikTok, Lemon8 ให้อยู่ในที่เดียวที่ search/filter/compare ได้

Success = นักศึกษาเปิด GIntern เป็นที่แรกเมื่อต้องตัดสินใจเรื่องที่ฝึกงาน

## Brand Personality

**Playful, Genuine, Friendly**

- ขี้เล่นแต่จริงใจ — ไม่ทางการ ไม่ corporate แต่ไม่เด็กเกินไป
- Safezone — ผู้ใช้รู้สึกปลอดภัยที่จะเขียนรีวิวจริงๆ
- เป็นกันเอง — เหมือนคุยกับรุ่นพี่ที่เคยผ่านมา ไม่ใช่อ่านรายงาน

## Anti-references

- **Glassdoor / JobsDB** — corporate เย็นชา, ดูเหมือนเว็บหางาน ไม่เป็นกันเอง, UI แน่น ไม่น่าอ่าน
- **เว็บราชการ / สถาบันการศึกษา** — โบราณ ไม่ทันสมัย ใช้ยาก ดูไม่น่าเชื่อถือในสายตาเด็กรุ่นใหม่
- ห้ามดู SaaS / fintech / enterprise ทุกรูปแบบ — GIntern ไม่ใช่เครื่องมือทำงาน

## Design Principles

1. **Safezone first** — ทุกอย่างต้องทำให้ผู้ใช้รู้สึกปลอดภัยที่จะแชร์ ตั้งแต่ tone of voice จนถึง UI ที่ไม่กดดัน
2. **Scroll-friendly, not task-heavy** — รีวิวเป็นคอนเทนต์ที่อ่านเพลินๆ เหมือน feed ไม่ใช่ฟอร์มที่ต้องเติมให้ครบ
3. **Show real voices** — ให้เสียงของผู้รีวิวเป็นพระเอก ไม่ใช่ UI element ที่แย่งความสนใจ
4. **Low friction, high value** — ทุก interaction ต้องง่าย ไม่มี step ที่ไม่จำเป็น ค้นหาเจอเร็ว เขียนรีวิวเสร็จเร็ว

## References

- **Fastwork** — playful pastel, visual identity จดจำได้ทันที, เป็นกันเอง
- **Lemon8 / Pinterest** — card-based feed, visual-first, สีสันสดใส, scroll เพลิน

## Accessibility & Inclusion

- WCAG AA — contrast 4.5:1 สำหรับ body text, 3:1 สำหรับ large text
- Keyboard navigation ใช้งานได้ครบ
- Reduced motion alternative สำหรับ animation ทุกจุด
- Font size อ่านง่ายบนมือถือ (base 16px+)
